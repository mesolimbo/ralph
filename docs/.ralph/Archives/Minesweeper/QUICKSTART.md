# Retro Minesweeper - Quick Start

## Play the Game

Simply open `/workspace/minesweeper.html` in your web browser!

## Game Controls

- **Left-click** - Reveal a cell
- **Right-click** - Place/remove a flag
- **Click smiley** - Restart game

## Objective

Find all 12 mines on the 10x10 grid without clicking on any mine. Numbers show how many mines are adjacent to that cell.
